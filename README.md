## Installation and usage

### Runing the project

Clone this repository: 

```
git clone https://github.com/juan18506/redux-toolkit-exercises.git
cd redux-toolkit-exercises
```

Install packages :

```
pnpm install
```

When installation is complete, run :

```
pnpm dev
```
